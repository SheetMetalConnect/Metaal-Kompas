*Configuratiebeheer* is een begrip binnen de industriële digitalisering en productstructuur & documentatie.

## Definitie

**Configuratiebeheer** is een systematische aanpak voor het identificeren, controleren en traceren van alle elementen die de configuratie van een product, systeem of proces bepalen. Het waarborgt dat de juiste versies van onderdelen, documentatie en software worden gebruikt en dat wijzigingen gecontroleerd worden doorgevoerd.

## Kenmerken

- **Configuratie-identificatie**: Unieke identificatie van alle product elementen en versies
- **Baseline management**: Gecontroleerde referentiepunten voor product configuraties
- **Change control**: Systematische beheersing van alle wijzigingen via [[engineering-change-order|ECO]]
- **Status accounting**: Real-time overzicht van alle configuratie elementen en hun status
- **Audit en verificatie**: Regelmatige controle op configuratie integriteit
- **[[traceerbaarheid-en-productidentificatie|Traceability]]**: Volledige history van configuratie wijzigingen
- **Versiecontrole**: Systematisch beheer van document en software versies
- **Access control**: Gecontroleerde toegang tot configuratie elementen

## Toepassing

**Product configuratiebeheer:**
- **[[engineering-bom|eBOM]] baseline**: Gecontroleerde engineering product structure
- **[[manufacturing-bom|mBOM]] synchronisatie**: Production configuration alignment met design
- **[[serienummer|Serial number]] tracking**: Individual product configuration identification
- **Spare parts management**: Configuration-specific replacement parts catalogs

**Metaalproduct configuraties:**
- **Material specifications**: Controlled steel grades, treatments, certifications
- **Machining configurations**: Tooling setups, fixture configurations, program versions
- **Welding configurations**: [[welding-procedure-specification|WPS]], electrode specifications, parameter sets
- **Assembly configurations**: Torque specifications, sealant types, hardware variants

**Software configuratiebeheer:**
- **CNC programma's**: [[cnc-programmatie|CNC]] program versioning en change control
- **Robot programma's**: [[offline-programmeren|OLP]] program baseline management
- **[[manufacturing-execution-system|MES]] configuraties**: Production route configurations
- **Quality system setup**: [[statistical-process-control-software|SPC]] parameter configurations

**Documentation configuratiebeheer:**
- **[[werkinstructie|Work instructions]]**: Controlled procedure versions
- **[[controleplan|Control plans]]**: Quality control configuration baselines
- **Test procedures**: Validation en verification procedure versions
- **Training materials**: Competency development content control

**Digital systems configuratiebeheer:**
- **[[enterprise-resource-planning|ERP]] setup**: System configuration parameters
- **[[product-lifecycle-management|PLM]] structures**: Product data organization schemes
- **Network configurations**: Industrial network topology management
- **Security configurations**: Access control en permission management

**Configuration management proces:**
- **Planning**: CM strategy development en tool selection
- **Identification**: Configuration item definition en numbering schemes
- **Control**: Change approval workflows en implementation procedures
- **Status accounting**: Configuration database maintenance en reporting
- **Audit**: Periodic verification van configuration integrity
