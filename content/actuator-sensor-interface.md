*Actuator‑Sensor Interface (AS‑Interface)* is een begrip binnen de industriële digitalisering en industriële communicatieprotocollen.

## Definitie

**Actuator-Sensor Interface (AS-Interface)** is een eenvoudig en kosteneffectief industrieel netwerk voor de laagste niveau van fabrieksautomatisering. Het protocol verbindt sensoren en actuatoren rechtstreeks met een controller via een simpele tweedraads-verbinding die zowel voeding als communicatie verzorgt.

## Kenmerken

- **Two-wire system**: Single cable voor both power en data communication
- **Simple topology**: Daisy-chain or tree structure without complex networking
- **Cost-effective**: Low-cost solution voor basic sensor/actuator connectivity
- **Easy installation**: Simple wiring with standard unshielded cables
- **Fast communication**: Cycle times as low as 5ms voor real-time control
- **Robust protocol**: Designed voor harsh industrial environments
- **Hot-swappable**: Devices can be replaced during operation
- **Diagnostic capabilities**: Built-in error detection en status monitoring
- **Limited data**: Optimized voor simple on/off en analog values

## Toepassing

**Manufacturing automation:**
- **Assembly lines**: Position sensors, limit switches on conveyor systems
- **Machine tools**: Tool position sensors, coolant level monitors
- **Robotics**: End effector sensors, safety light curtains
- **Material handling**: Presence detection, weight sensors on conveyors

**Safety systems:**
- **Emergency stops**: Distributed e-stop buttons throughout facility
- **Safety doors**: Position switches on machine guards en access panels
- **Light curtains**: Safety beam systems voor personnel protection
- **Two-hand controls**: Operator safety controls requiring dual activation

**Process monitoring:**
- **Temperature sensors**: Simple thermal monitoring in production areas
- **Pressure switches**: Pneumatic system monitoring en control
- **Flow sensors**: Basic liquid en gas flow detection
- **Level sensors**: Tank en hopper level monitoring

**Building systems:**
- **HVAC control**: Temperature sensors, damper actuators
- **Lighting control**: Occupancy sensors, light level monitoring
- **Access control**: Door sensors, card readers
- **Fire safety**: Smoke detectors, alarm indicators

**Metal industry applications:**
- **Welding stations**: Gas flow sensors, electrode position monitoring
- **Cutting systems**: Torch height sensors, material presence detection
- **Heat treatment**: Furnace door position, temperature monitoring
- **Quality control**: Go/no-go gauging, part presence verification

**Network integration:**
- **Fieldbus gateways**: Connection to PROFIBUS, DeviceNet, Ethernet
- **SCADA integration**: Data collection voor higher-level monitoring
- **HMI connectivity**: Operator interface displays van AS-i status
- **Maintenance systems**: Diagnostic data voor predictive maintenance
